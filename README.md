LoL Account SwitcherLoL Account Switcherは、League of Legends (Riot Client) の複数アカウントを管理し、シームレスに切り替えるためのWindowsシステムトレイ常駐型アプリケーションです。  

煩わしいログアウト・ログイン作業を省略し、セッションファイルのバックアップと復元、およびログイン画面での自動キー入力（マクロ）を組み合わせることで、ワンクリックで目的のアカウントへログインします。 

主な機能
高速アカウント切り替え: Riot Clientのセッション情報をローカルにプロファイルとして保存し、次回以降はファイルを入れ替えるだけでログイン状態を復元します。 

自動ログイン（マクロ機能）: セッションが期限切れの場合や新規アカウントの追加時、Riot Clientのログイン画面にてIDとパスワードを自動入力します。  

ゲームの自動起動: バックグラウンドでローカルAPIを監視し、ログイン完了を検知すると自動的にLeague of Legendsを起動します。  （動作不安定）

ランク情報の自動取得: Riot APIを利用し、登録アカウントのSolo/Duoランクを取得してメニュー上に表示します。  PUUID変換ツール: Riot ID (Name#TAG) と PUUID を相互変換するツールを内蔵し、検索履歴機能も備えています。  

多重起動防止: アプリケーションの複数同時起動を防ぐフェイルセーフ機能が組み込まれています。  

動作環境とセットアップ対応OS: Windows 10 / 11必須ソフトウェア: League of Legends および Riot Client がインストールされていること。

APIキーの設定（オプション）
ランク情報の取得およびPUUID変換ツールを使用するには、Riot Developer Portalで取得したAPIキーが必要です。
実行ファイル（.exe）と同じディレクトリ、または %LOCALAPPDATA%\Switch\ フォルダに .env という名前のファイルを作成し、以下のように記述してください。  
RIOT_API_KEY="RGAPI-xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx"

使い方
実行ファイルを起動すると、タスクバーのシステムトレイにアイコンが常駐します。 
左クリック: 登録済みアカウントの一覧を表示します。アカウント名をクリックすると、そのアカウントへの切り替え処理が開始されます。 アカウントを切り替えるときにサインアウトしてしまうとログインセッションが無効化されるため、ワンクリック切り替えができずマクロ切り替えになります。
アカウント切り替えを毎回本ツールから行えばログインされた状態でクライアントが軌道します。
サインアウトしてしまった場合は自動でIDとPWの自動入力モードに切り替わるのでenterを押してマクロを起動してログインしてください。マクロログインでセッションを再度保存するので次からはワンクリックで切り替え可能になります。
右クリック: メニュー（PUUIDツール、設定ファイルを開く、終了）を表示します。  

アカウントの新規追加手順トレイアイコンを左クリックし、「[+] アカウントを追加」を選択します。 
ログインIDとパスワードを入力して保存します。  既存のRiot関連プロセスが終了し、Riot Clientが自動的に起動します。 
ログイン画面が表示され、入力可能状態になったらキーボードの Enter キー（デフォルトのトリガーキー）を押します。  
マクロが作動してIDとパスワードが自動入力され、ログイン完了後にセッション情報が保存されます。  
アカウントの編集・削除
左クリックメニューに表示されているアカウント名を右クリックすると、登録情報の変更や削除を行えます。  
内部仕様とデータ保存先本アプリケーションはレジストリを使用せず、以下のディレクトリにデータを保存します。  
ベースディレクトリ: %LOCALAPPDATA%\Switch  
設定ファイル: %LOCALAPPDATA%\Switch\settings.json  
保存される情報: アカウントのログイン情報（平文）、Riot Clientのパス、マクロのトリガーキー、PUUIDの検索履歴など。  
プロファイル保存先: %LOCALAPPDATA%\Switch\Profiles\<Riot ID>  
切り替え時に以下のファイルやフォルダを Riot Client のディレクトリと入れ替えます。  
RiotGamesPrivateSettings.yaml  Sessions フォルダ  RiotClientSettings.yaml  

注意事項セキュリティ: 入力したログインIDとパスワードは、ローカルの settings.json に暗号化されずにそのまま保存されます。PCの共有やファイルの取り扱いには十分ご注意ください。  マクロ実行時の操作: マクロ（自動入力）の動作中はマウスやキーボードを操作しないでください。入力ズレなどの誤動作の原因となります。

利用規約に関する免責: 本アプリケーションはRiot Gamesの公式ツールではありません。クライアントファイルの操作やサードパーティツールの使用に伴うアカウントへのペナルティ等について、開発者は一切の責任を負いません。APIキーの有効期限: 開発用のPersonal API Keyを使用している場合、24時間で期限が切れるため、定期的に .env ファイル内のキーを更新する必要があります。期限が切れるとランク取得機能などは動作しません。
