# LoL Account Switcher

[cite_start]タスクトレイに常駐し、League of Legendsの複数アカウントをシームレスに切り替えるためのサポートツールです [cite: 88]。

## Features

* [cite_start]**アカウントの高速切り替え:** 保存したセッションデータを利用して、ワンクリックで別アカウントへログイン・クライアント起動を行います [cite: 88]。
* [cite_start]**アカウントの自動登録:** 新規追加時は、自動マクロによってID/PWを入力し、セッションを安全に保存します [cite: 88]。
* [cite_start]**現在ランクの自動表示:** アカウント一覧のRiot IDの横に、現在のランク（例: `G4`、`un`）を自動で取得して表示します [cite: 88]。
* [cite_start]**PUUID ⇄ Riot ID 変換ツール:** APIを利用して、IDからPUUID、PUUIDからIDを相互に検索・履歴保存できます [cite: 88]。

## Getting Started

[cite_start]本ツールの一部機能（ランク表示、PUUIDツール）を利用するには、Riot Developer Portalから取得したAPIキーが必要です [cite: 89][cite_start]。以下の手順で設定ファイルを必ず作成してください 。

1. [cite_start]**APIキーの取得:** Riot Developer Portalにログインし、テスト用のAPIキー（`RGAPI-` から始まる文字列）をコピーします 。
2. [cite_start]**`.env` ファイルの作成:** メモ帳を開き、以下の一文のみを貼り付けます（空白や `""` などの記号は不要です） 。
   ```env
   RIOT_API_KEY=RGAPI-xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx

   ファイルの配置: 作成した .env ファイルは、本ツールの実行ファイル（.exe）と全く同じフォルダ内に置いてください 。


タスクトレイアイコンを【左クリック】

アカウントの切り替え: 一覧からアカウント名をクリックすると、自動でLoLが起動します 。
セッションが無効化されていて自動ログインできないときはRIOTクライアント起動後エンターを押すとマクロが起動しログインします。


アカウントの追加: 「[+] アカウントを追加」からIDとPWを入力すると、マクロが作動して情報を登録します 。


情報の変更・削除: 一覧のアカウント名を【右クリック】すると、パスワードの変更やリストからの削除が可能です 。

タスクトレイアイコンを【右クリック】

PUUIDツール: IDやPUUIDの相互変換を行うツールを開きます 。


設定 (メモ帳): 内部データが保存されている settings.json を直接開きます 。


終了: ツールを完全に終了させます 。

Troubleshooting
Q. 昨日まで見えていたランクが更新されない / PUUIDツールが動かない


A. Riotのテスト用APIキーは取得から24時間で無効（有効期限切れ）になります 。Developer Portalで新しくキーを再発行し、.env ファイルの中身を新しいものに書き換えてください 。

Q. 初回起動時からランクが出ない / PUUIDが機能しない


A. .env ファイルの読み込みに失敗しています 。ファイル名が .env.txt や env になっていないか、Windowsの「ファイル名拡張子」を表示する設定にして確認してください 。

Q. アカウント追加時の自動入力（マクロ）が失敗する


A. ツールが入力待ち状態になったら、画面の指示に従って指定のキー（デフォルトはEnter）を押してください 。入力中にマウスやキーボードを触ると誤作動の原因になります 。大文字小文字などにも気を付けてください
